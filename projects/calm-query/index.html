Calm Query

A small internal tool for scanning backend changes and release notes

AstroTypeScriptStatic hosting Source

Calm Query is less a product and more a constraint exercise: what does a readable engineering site look like when you remove everything that is not essential?

Problem

The site needed to present technical work without turning into a generic portfolio template. That meant avoiding oversized hero imagery, marketing language, and decorative noise.

Architecture

The implementation uses a static Astro build, content collections, and small shared layout pieces. The visual language leans on spacing, borders, and a restrained palette rather than heavy effects.

Highlights

  • A home page that behaves like a portal instead of a billboard
  • Separate list and detail routes for blog and projects
  • Prose styling that keeps articles comfortable on desktop and mobile
